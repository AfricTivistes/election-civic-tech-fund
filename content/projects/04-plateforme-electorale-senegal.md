## Outils déployés

**Vie Publique Sénégal** opère la plateforme [vie-publique.sn](https://www.vie-publique.sn/), un outil de référence pour la transparence électorale au Sénégal.

### Vie-Publique.sn

La plateforme agrège les données électorales officielles et les rend accessibles en open data. Elle inclut une cartographie interactive des bureaux de vote, les listes de candidats, et les résultats en temps réel lors des scrutins.

### OCR et traitement documentaire

Un système OCR automatise l'extraction de données depuis les procès-verbaux officiels publiés en format PDF, les transformant en données structurées réutilisables.

### API ouverte

Les développeurs et chercheurs peuvent accéder aux données via une API REST documentée, permettant la création d'applications tierces basées sur les données électorales sénégalaises.

### Code source

L'ensemble du projet est open source et disponible sur GitHub, permettant à d'autres pays africains de s'approprier et d'adapter l'outil.
