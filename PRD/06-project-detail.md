# PRD 06 - Page Détail Projet

## Objectif

Présenter chaque projet bénéficiaire de manière complète avec toutes les informations pertinentes, sa progression, et ses actualités spécifiques, tout en respectant la confidentialité (pas d'email affiché).

## URL

- `/fr/projects/[id]` (ex: `/fr/projects/1`)
- `/en/projects/[id]`

## Génération statique

```typescript
// app/[lang]/projects/[id]/page.tsx

export async function generateStaticParams() {
  const projects = getAllProjects()
  
  return projects.flatMap(project => [
    { lang: 'fr', id: project.id },
    { lang: 'en', id: project.id }
  ])
}

interface ProjectPageProps {
  params: Promise<{ lang: string; id: string }>
}

export default async function ProjectPage({ params }: ProjectPageProps) {
  const { lang, id } = await params
  const project = getProjectById(id)
  const projectNews = await getNewsByProject(id, lang)
  const similarProjects = getSimilarProjects(id, 3)

  )
}
```

## SEO

```typescript
export async function generateMetadata({ params }: ProjectPageProps) {
  const { lang, id } = await params
  const project = getProjectById(id)
  
  return {
    title: `${project.projectName[lang]} | Election Civic Tech Fund`,
    description: project.description[lang].slice(0, 160),
    openGraph: {
      title: project.projectName[lang],
      description: project.description[lang],
      images: [project.projectImage]
    }
  }
}
```

## Critères d'acceptation

- [ ] La page affiche toutes les informations du projet
- [ ] La barre de progression est visible avec dates
- [ ] Le budget alloué est affiché clairement
- [ ] Les technologies sont listées avec icônes
- [ ] **Aucun email n'est affiché**
- [ ] Le lien vers le site web fonctionne (si disponible)
- [ ] Les actualités du projet sont listées
- [ ] 3 projets similaires sont suggérés
- [ ] Le breadcrumb permet de revenir à la liste
- [ ] La page est générée statiquement
- [ ] Les métadonnées SEO sont correctes
- [ ] La page est responsive
